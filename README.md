# Building from scratch
## 1. Install Unity 2022.3.18f1
You can either use [Unity Hub](unityhub://2022.3.18f1/d29bea25151d) or [download the installer manually](https://unity.com/releases/editor/whats-new/2022.3.18f1#installers).
Building for Quest 2/3 requires the [Android SDK](https://docs.unity3d.com/2022.3/Documentation/Manual/android-sdksetup.html).

## 2. Clone this repository
```
git clone https://github.com/aitorelrincon/projectGroup5.git
```

## 3. Open the project


## 4. Build the project
Build your [PC executable](https://docs.unity3d.com/2022.3/Documentation/Manual/WindowsStandaloneBinaries.html) or [Quest 2/3 apk](https://docs.unity3d.com/2022.3/Documentation/Manual/android-BuildProcess.html).
